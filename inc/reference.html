DSQ Slick Slider

Source code can be found here: http://kenwheeler.github.io/slick/
The javascript & css is already taken care of by the plugin. CSS can be added to the regular css files of the theme, or the Slick Slider Plugin may be edited directly. Add the following HTML to a page to implement a Slick Slider instance.

<div class="dsq-slick-slide">
  <div>your content</div>
  <div>your content</div>
  <div>your content</div>
</div>

jQueryUI Accordions & Tabs Usage

Accordions: Use #jqui_accordion if a single accordion or #jqui_accordion-(n) if multiple accordions.
Tabs: Use #jqui_accordion if a single accordion or #jqui_accordion-(n) if multiple accordions.

Accordion Markup: 
<div id="jqui_accordion">
  <h3>Section 1</h3>
  <div>
    <p>
    Content.
    </p>
  </div>
  <h3>Section 2</h3>
  <div>
    <p>
      Content.
    </p>
  </div>
</div>
Tabs Markup: 
<div id="jqui_tabs">
  <ul>
    <li><a href="#tabs-1">Nunc tincidunt</a></li>
    <li><a href="#tabs-2">Proin dolor</a></li>
    <li><a href="#tabs-3">Aenean lacinia</a></li>
  </ul>
  <div id="tabs-1">
    <p>Content</p>
  </div>
  <div id="tabs-2">
    <p>Content.</p>
  </div>
  <div id="tabs-3">
    <p>Content</p>
  </div>
</div>
